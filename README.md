# Coding Starter Kit

## Features

- Sass(.scss)
- ES2015+ (Babel)
- pug

### Tool
- webpack 2
- gulp
- editorconfig

## Install

```
$ git clone https://d-kusk@bitbucket.org/ficckyoto/coding-starter-kit.git
```

## Usage

```
$ npm install

$ npm start
```

### Optional commands

```
$ npm run sass  // Compiling for Sass only

$ npm run js  // Compiling for JavaScript(ES2015) only
```

Look at package.json
