# Coding Starter Kit

## Features

- Sass(.scss)
    + Bourbon
    + Neat
- ES2015+ (Babel)
- pug

### Tool
- webpack 2
- gulp
- editorconfig
- [Yarn](https://yarnpkg.com/en/)

## Install

```
$ git clone https://d-kusk@bitbucket.org/ficckyoto/coding-starter-kit.git
```

## Usage

```
$ yarn install

$ yarn start
```

### Optional commands

```
$ yarn run sass  // Compiling for Sass only

$ yarn run js  // Compiling for JavaScript(ES2015) only
```

Look at package.json
